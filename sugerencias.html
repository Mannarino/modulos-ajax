Modulo de sugerencias


Input de buscador con sugerencias construido con Ajax.
Este sistema esta construido con ajax, del lado del servidor utilice php y el gestor de base de datos mysql.
Tambien cuenta con un desarrollo en javascript para manejar diferentes situaciones que pasan con el input, como por ejemplo un soporte de teclado para que se maneje las opciones de sugerencia que aparecen con las flechas del teclado.
