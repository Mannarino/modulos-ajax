Este sitio tiene 3 modulos construidos con Ajax.

Un modulo con un input donde se evalua si el usuario que se quiere registrar esta ya registrado o no, realizando una consulta
a la base de datos en tiempo real para la comprobacion y devolviendole al usuario la informacion de disponibilidad o no.

Otro modulo que es un mini chat.

Y un modulo donde se encuentra un input de un buscador donde aparecen sugerencias de busqueda en tiempo real.

Para los tres modulos use del lado del servidor el lenguaje php, y el gestor de base de datos mysql.
Utilice javascript para hacer las consultas asincronicas y para la comprobacion de formulario y otros manejos de eventos.
La maquetacion esta hecha en html5 y utilice ccs3 para diferentes efectos visuales como border radius , sombras, etc.
